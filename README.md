# Mentor API (Node.js + Express + MVC)

A simple RESTful API for managing mentors, built with **Node.js**, **Express**, and **MongoDB**, following the **MVC architecture**.

This project allows you to **Create, Read, Update, and Delete (CRUD)** mentors.

## Features

- Create, read, update, and delete mentors
- Organized using **MVC pattern** for scalability and maintainability
- RESTful API endpoints
- Easy to integrate with frontend applications


## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose (ORM)
- JavaScript

---

## Installation

## 1. Install dependencies
npm install

## 2.Run the Project
npm start
The server will run on http://localhost:3000
API endpoints are under /api/mentors
